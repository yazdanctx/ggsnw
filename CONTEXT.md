# ggsnw — Domain Glossary

## Core Concepts

### Shortname
A truncated 8.3-style IIS shortname (e.g. `ADMIN~1`, `WEB~1.CON`). The input to the tool. In AI mode, the `~` suffix is stripped before querying.

### Real name
The full filename extracted from a GitHub Code Search result whose name contains the shortname (case-insensitive `filename:` query). One type of output.

### AI guess
A predicted full name generated by OpenAI based on the shortname prefix (e.g. `ADMIN` → `AdminPage`). Used in AI mode.

### Wordlist
The set of unique words (real names or AI guesses) collected across all searched shortnames, deduplicated.

## Modes

### GitHub mode (default)
Searches GitHub Code Search for real filenames matching the shortname.

### AI mode
Uses OpenAI to generate plausible full-name guesses for each shortname. Configurable guess count via `--guesses`.

## Tool Behaviour

### Input modes
- Single shortname: positional argument (`ggsnw ADMIN~1`)
- Shortname list file: `-f shortnames.txt` (one per line)

### Output format
- Live during search: `\t[x] {realname}` printed per previously-unseen match
- No-match per shortname: `\t[ ] {shortname} not found`
- Final summary: `-----` border, one realname per line, `-----` border
- Optional file output: `-o wordlist.txt`

### Token persistence
- GitHub token: `ggsnw --token ghp_xxx` or interactive prompt
- OpenAI key: `ggsnw --key sk-xxx` or interactive prompt in AI mode
- Stored at: `~/.config/ggsnw/config.json` (`github_token` and `openai_key` fields)
- Subsequent runs: tokens read automatically

### API behaviour
- Uses GitHub Code Search API (authenticated) in GitHub mode
- Uses OpenAI Chat Completions API in AI mode
- Full pagination per query (up to 1000 results max, GitHub-enforced limit)
- Rate-limit aware: respects `X-RateLimit-Remaining`, sleeps when near empty
- No parallelism; sequential per-shortname

## Limits

| Limit | Value |
|---|---|
| Max results per query | 1000 (10 pages × 100) |
| GitHub rate limit (authenticated) | 30 req/min |
| AI model | gpt-3.5-turbo |
| AI guesses per shortname | Configurable via `--guesses` (default 10) |
| Match mode (GitHub) | Case-insensitive, filename segment |
| Suffix stripping (AI) | Remove `~...` before querying |
